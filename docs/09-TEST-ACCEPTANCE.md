## 3. 核心测试用例

### 3.1 用例1：用户观看视频完整流程

**前置条件**：
- 已有视频数据
- 用户已注册

**步骤**：
1. 用户打开首页
2. 点击分类"电影"
3. 点击第一个视频
4. 点击播放按钮
5. 选择"蓝光"线路
6. 观看5分钟后退出
7. 重新进入该视频

**预期结果**：
- 步骤3：正确显示视频详情页
- 步骤4：播放器正常加载
- 步骤5：视频正常播放
- 步骤7：从上次位置续播

### 3.2 用例2：采集任务执行

**前置条件**：
- 已配置采集规则

**步骤**：
1. 管理员进入采集管理
2. 点击"执行采集"
3. 等待采集完成
4. 查看采集日志
5. 前台查看新增视频

**预期结果**：
- 步骤3：采集成功，无报错
- 步骤4：日志显示采集数量
- 步骤5：前台显示新视频

### 3.3 用例3：VIP权限控制

**前置条件**：
- 有VIP视频
- 有普通用户和VIP用户

**步骤**：
1. 普通用户打开VIP视频
2. 尝试播放
3. VIP用户打开同一视频
4. 尝试播放

**预期结果**：
- 步骤2：提示"VIP专享"，无法播放
- 步骤4：正常播放
