# 产品需求文档 PRD

## 1. 功能总览

| 功能模块 | 描述 | 优先级 | 备注 |
|---------|------|--------|------|
| 视频管理 | 视频的增删改查、多播放源、分集管理 | P0 | 核心功能 |
| 文章管理 | 图文内容的发布与管理 | P1 | 内容补充 |
| 分类标签 | 多级分类、标签系统 | P0 | 内容组织 |
| 采集系统 | 兼容苹果CMS采集规则 | P0 | 内容来源 |
| 播放器 | 多线路切换、记忆播放 | P0 | 用户体验 |
| 用户系统 | 注册登录、收藏历史 | P0 | 用户留存 |
| 搜索系统 | 全文搜索、筛选过滤 | P1 | 内容发现 |
| 管理后台 | 内容管理、系统配置 | P0 | 运营工具 |

## 2. 用户故事

### 站长（管理员）

- 作为站长，我希望快速搭建影视站点，从而快速上线运营
- 作为站长，我希望兼容苹果CMS采集规则，从而复用现有采集源
- 作为站长，我希望管理视频内容，从而维护站点内容质量
- 作为站长，我希望查看数据统计，从而了解站点运营情况

### 普通用户

- 作为用户，我希望快速找到想看的视频，从而节省时间
- 作为用户，我希望视频播放流畅，从而获得良好体验
- 作为用户，我希望记录播放进度，从而方便续看
- 作为用户，我希望收藏喜欢的视频，从而方便回看

## 3. 功能细化

### 3.1 视频管理

**输入**：
- 视频标题、副标题、简介
- 封面图片（URL或上传）
- 分类（多级）
- 标签（多个）
- 播放源（多个，支持m3u8/mp4/flv）
- 分集信息（可选）

**输出**：
- 视频详情页
- 视频列表
- 播放页

**触发条件**：
- 手动添加
- 采集导入
- API导入

**成功结果**：
- 视频保存成功
- 分类标签关联正确
- 播放源可正常播放

**失败结果**：
- 必填字段缺失
- 播放源格式错误
- 分类不存在

**权限要求**：
- 管理员：全部操作
- 编辑：添加、编辑（无删除权限）

**边界条件**：
- 标题长度：2-100字符
- 简介长度：0-2000字符
- 标签数量：0-10个
- 播放源数量：1-10个

### 3.2 采集系统

**输入**：
- 采集规则（XML/JSON格式，兼容苹果CMS）
- 目标URL
- 采集参数（分页、筛选等）

**输出**：
- 采集到的视频数据
- 采集日志

**触发条件**：
- 定时任务触发
- 手动触发
- API调用

**成功结果**：
- 数据解析成功
- 去重正确
- 图片下载成功
- 分类匹配正确

**失败结果**：
- 规则解析失败
- 网络请求失败
- 数据格式不匹配

**权限要求**：
- 仅管理员可操作

**边界条件**：
- 单次采集数量限制：100条
- 采集频率限制：最小间隔5分钟

### 3.3 播放器

**输入**：
- 视频ID
- 播放源URL
- 用户ID（用于记录进度）

**输出**：
- 视频播放
- 播放进度
- 播放统计

**触发条件**：
- 用户点击播放
- 自动下一集

**成功结果**：
- 视频正常播放
- 进度正确记录
- 多线路可切换

**失败结果**：
- 播放源失效
- 网络错误
- 格式不支持

**权限要求**：
- 普通用户：播放公开视频
- VIP用户：播放VIP视频

**边界条件**：
- 支持格式：m3u8, mp4, flv
- 最大清晰度：4K
- 记忆时长：30天

## 4. 业务规则

- 视频标题必须唯一
- 分类下必须有内容才能显示
- 采集数据必须去重（按标题+年份）
- VIP视频仅VIP用户可播放
- 播放源失效自动隐藏

## 5. 状态变化

### 视频状态

- **初始状态**：草稿
- **中间状态**：审核中
- **成功状态**：已发布
- **失败状态**：已下架

### 采集任务状态

- **初始状态**：待执行
- **中间状态**：执行中
- **成功状态**：已完成
- **失败状态**：失败（可重试）

## 6. 埋点 / 日志 / 运营要求

- 视频播放量统计
- 用户播放时长统计
- 采集成功率统计
- 搜索热词统计
- 用户留存率统计
