## 2. 安全

### 2.1 认证与鉴权

- **认证方式**：JWT Token
- **Token有效期**：7天
- **刷新机制**：自动刷新（提前1天）
- **密码策略**：最小8位，包含字母+数字
- **登录失败**：5次失败后锁定15分钟

### 2.2 权限模型

- **模型**：RBAC（基于角色的访问控制）
- **角色**：
  - 超级管理员：全部权限
  - 内容管理员：视频/文章管理
  - 运营人员：查看数据，无修改权限

### 2.3 数据加密

- **密码**：bcrypt加密存储
- **敏感数据**：AES加密
- **传输**：HTTPS全站

### 2.4 审计日志

- **记录内容**：谁、何时、做了什么、结果
- **保留时间**：90天
- **存储**：独立日志库

### 2.5 防护措施

| 攻击类型 | 防护措施 |
|---|---|
| SQL注入 | 参数化查询，ORM防护 |
| XSS | 输入过滤，输出转义，CSP |
| CSRF | Token验证，SameSite Cookie |
| 文件上传 | 类型检查，大小限制，病毒扫描 |
| DDoS | 限流，CDN防护，WAF |

## 3. 可用性

### 3.1 可用性目标

- **SLA**：99.9%
- **年停机时间**：< 8.76小时
- **故障恢复时间**：< 30分钟
- **回滚时间**：< 15分钟

### 3.2 备份策略

- **数据库**：每日全量备份，保留7天
- **文件存储**：对象存储多副本
