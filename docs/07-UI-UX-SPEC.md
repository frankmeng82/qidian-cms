## 2. 页面级说明

### 2.1 首页

- **页面目标**：吸引用户，快速找到感兴趣的内容
- **入口**：直接访问
- **核心组件**：
  - 顶部导航（Logo、分类、搜索、用户入口）
  - Banner轮播（推荐内容）
  - 分类区块（电影、电视剧、综艺等）
  - 热门排行
  - 最新更新
- **状态**：加载中、加载完成、空态
- **操作**：点击视频进入详情、点击分类进入分类页、搜索
- **空态**：显示"暂无内容，敬请期待"
- **错误态**：显示"加载失败，点击重试"
- **加载态**：骨架屏加载

### 2.2 详情页

- **页面目标**：展示视频详细信息，引导用户播放
- **入口**：首页、分类页、搜索页
- **核心组件**：
  - 视频封面（大图）
  - 视频标题、副标题
  - 视频信息（年份、地区、类型、导演、演员）
  - 简介
  - 播放按钮
  - 分集列表（电视剧）
  - 相关推荐
- **状态**：已登录/未登录、VIP/非VIP
- **操作**：点击播放、收藏、分享
- **空态**：404页面
- **错误态**：视频不存在提示
- **权限限制**：VIP视频显示"VIP专享"遮罩

### 2.3 播放页

- **页面目标**：提供流畅的播放体验
- **入口**：详情页点击播放
- **核心组件**：
  - 播放器（Video.js）
  - 线路选择
  - 清晰度选择
  - 播放控制（播放/暂停、进度、音量、全屏）
  - 下一集按钮
  - 视频信息
- **状态**：播放中、暂停、加载中、错误
- **操作**：播放控制、线路切换、清晰度切换、全屏
- **空态**：无
- **错误态**：播放失败，显示"视频加载失败，切换线路重试"
- **加载态**：加载动画

### 2.4 管理后台-视频管理

- **页面目标**：高效管理视频内容
- **入口**：独立后台地址
- **核心组件**：
  - 视频列表表格
  - 搜索/筛选
  - 分页
  - 添加/编辑/删除按钮
  - 批量操作
- **状态**：加载中、空态
- **操作**：增删改查、批量操作
- **空态**：显示"暂无视频，点击添加"

## 3. 交互规则

### 3.1 通用交互

- **点击反馈**：所有可点击元素有hover效果
- **加载状态**：异步操作显示loading
- **错误提示**：Toast提示，3秒自动消失
- **确认操作**：删除等危险操作需二次确认
- **表单验证**：实时验证，错误提示在字段下方

### 3.2 播放器交互

- **自动播放**：进入播放页自动播放（需用户授权）
- **记忆播放**：从上次进度续播
- **快捷键**：空格播放/暂停，方向键控制进度/音量
- **画中画**：支持画中画模式

## 4. 响应式要求

### 4.1 Desktop（≥1200px）

- 导航：顶部固定导航
- 首页：多列布局，展示更多内容
- 播放器：大尺寸，右侧推荐

### 4.2 Tablet（768px-1199px）

- 导航：简化导航，汉堡菜单
- 首页：双列布局
- 播放器：全屏宽度

### 4.3 Mobile（<768px）

- 导航：底部Tab导航
- 首页：单列布局
- 播放器：全屏播放，自动横屏
- 详情页：信息折叠

## 5. 设计约束

### 5.1 品牌风格

- **品牌色**：#1890ff（Ant Design主色）
- **辅助色**：#52c41a（成功）、#faad14（警告）、#f5222d（错误）
- **中性色**：#000000-#ffffff灰度

### 5.2 字体

- **中文**：PingFang SC、Microsoft YaHei
- **英文**：-apple-system、BlinkMacSystemFont
- **代码**：Consolas、Monaco

### 5.3 颜色

- **背景**：#f0f2f5（页面背景）、#ffffff（卡片背景）
- **文字**：#000000（主标题）、#333333（正文）、#666666（次要）
- **边框**：#d9d9d9

### 5.4 动效

- **过渡时间**：200ms
- **缓动函数**：ease-in-out
- **加载动画**：Ant Design Spin

### 5.5 可访问性

- 支持键盘导航
- 图片添加alt属性
- 颜色对比度符合WCAG标准
- 支持屏幕阅读器

## 6. 简单测试前端

为快速验证，先实现简化版前端：

### 6.1 简化版页面

| 页面 | 功能 | 说明 |
|---|---|---|
| 首页简化版 | 视频列表、分类筛选 | 无轮播、无推荐算法 |
| 详情页简化版 | 视频信息、播放按钮 | 无相关推荐 |
| 播放页简化版 | 基础播放功能 | 单线路、无记忆播放 |
| 管理后台简化版 | 视频CRUD | 无权限控制、无批量操作 |

### 6.2 技术栈

- **框架**：React 18 + Vite
- **UI库**：Ant Design
- **播放器**：Video.js
- **路由**：React Router
- **状态管理**：React Hooks（简化版不使用Redux）

### 6.3 快速启动

```bash
# 创建项目
npm create vite@latest qidian-cms-web -- --template react-ts

# 安装依赖
cd qidian-cms-web
npm install antd video.js react-router-dom axios

# 启动开发
npm run dev
```
